# 前端 Demo 开发计划

## 项目概述

- **项目名称**：垂域模型评估系统前端演示 Demo
- **项目目标**：为产品经理提供可交互的演示系统，展示系统核心功能
- **技术选型**：React + Ant Design + Three.js + ECharts

---
